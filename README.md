# pyFetch
An attempt at a cross-platform version of the popular "screenFetch" script

#### Supports
* Ubuntu 16.04
* Windows 7/8/10

## Want your OS added?
Open an issue with your OS, and preferably provide a link to an ascii image of the logo :D
